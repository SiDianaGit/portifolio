# 🚀 Portfólio Profissional: Engenharia de Dados & IA

Bem-vindo ao repositório da minha página pessoal! Este projeto é uma Single Page Application (SPA) desenvolvida com **Python** e **Streamlit**. Ele funciona como um hub central para a minha carreira, apresentando minha trajetória, stack tecnológica e projetos (como o Bússola AI e automações cloud) em um formato minimalista e de alta performance.

## 🎯 Visão Geral
O design foi construído seguindo uma estética *Dark Mode* moderna (estilo Linear), focado em clareza, responsividade (Mobile-first) e conversão rápida para recrutadores e membros da comunidade técnica.

### ✨ Principais Funcionalidades
* **Hero Section Impactante:** Apresentação direta da proposta de valor com botões de chamada para ação (Ver Projetos, Download CV).
* **Vitrine de Projetos (Bento Grid):** Cards interativos destacando os principais projetos, com tags visuais para a stack utilizada (ex: Python, LangChain, Azure) e links diretos para repositórios e demos.
* **Linha do Tempo (Experiência):** Histórico cronológico inverso detalhando vivências em Engenharia de Dados, MLOps, e Business Intelligence.
* **Tech Stack Visual:** Seção organizada com as principais ferramentas, linguagens e plataformas em nuvem que utilizo no dia a dia.
* **Contato Integrado:** Formulário funcional para recebimento de mensagens diretas.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3.x
* **Framework Web:** Streamlit
* **Estilização:** Custom CSS embutido (Tipografia Inter, Dark Theme, Bento Grid UI)
